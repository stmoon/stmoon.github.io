Setting up a Ghost Blog with Github Pages

06 March 2016 on ghost, page

Install

  • npm 설치

How to setting with github

# (in a new tab) get back to the blog folder
$ cd ~/Dropbox/blog
$ buster setup --gh-repo=\
"https://github.com/<username>/<username>.github.io.git"
$ buster generate
$ buster deploy
  • 나중에 다시 연결 시에는 로컬에서 서버 시작하고, generate와 deploy 하면 된다.
$ npm start


$ buster generate
$ buster deploy
